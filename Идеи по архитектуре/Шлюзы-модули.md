Уже сделано основание.

Модуль - простой префаб с коллизией, [[Триггеры|триггером]], а также скриптами шлюзов.

Один шлюз - один скрипт, который описывает положение шлюза относительно центра родительского объекта, в данном случае модуля. Модуль становиться модулем, когда у него появляется хотя бы один шлюз.

Соединение модулей сейчас реализовано по шлюзам. Мы передаём в метод одного шлюза другой шлюз и один из модулей трансформируется подстраиваясь под другой модуль. Если он при этом был соединён с другим модулем, то связь скорее всего упадёт. Т.е. модули пока статичны после стыковки.

Ещё нет коллизии при вставке, т.е. нет проверки на возможность установки модуля, можно поставить "модуль в модуль".

#### Что планируется
Проверка не коллизирует ли модуль при вставке с другими.

Каждый модуль будет хранить информацию о себе. Помимо шлюзов это будут: тип модуля, его прочность, потребление энергии, а для некоторых запас энергии в аккумуляторах / обеспечение электричеством станции, запас еды.
