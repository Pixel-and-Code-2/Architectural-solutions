Заказ - торговая площадка, где деньгами является [[Влияние]]. За него можно получать другие метрики. Заказ нужно ждать, т.е. он не сразу приходит. ЧП будет провоцировать игрока заказать. Один заказ стоит какое-то количества влияния сразу, т.е. выгоднее будет заказывать пачками.

По архитектуре это вопрос больше интерфейса, по которому ещё мало информации.
