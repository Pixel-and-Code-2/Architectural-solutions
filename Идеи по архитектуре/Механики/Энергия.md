Есть ряд требований. А так это обычная метрика:
- Вырабатывается только днём. Ночью тратятся запасы.
- Аккумуляторы только в инженерном и базовом модуле.
- Для экономии энергии можно отключать модули.
- Каждый модуль по чуть-чуть кушает энергии.
- Запас энергии виден только, когда игрок трогает один из инженерных модулей.
- Настройка модуля пешкой занимает время.
